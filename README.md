# 🎂 Deliciu – Site despre torturi

**Deliciu** este un site dedicat pasionaților de torturi. Aici poți:
- Descoperi rețete de torturi delicioase
- Personaliza și comanda propriul tort
- Afla mai multe despre ingredientele folosite
- Vizualiza colecții de torturi pentru diferite evenimente

## 📁 Structura site-ului

- `index.html` – pagina principală
- `creeaza.html` – pagina unde îți creezi propriul tort
- `despre.html` – informații despre proiect
- `livrare.html` – detalii despre transport și livrare
- `contact.html` – formular de contact
- `style.css` – stilul general al site-ului
- `script.js` – funcționalități interactive (coș, salvări, butoane etc.)

## 🚀 Tehnologii folosite

- HTML5
- CSS3
- JavaScript (vanilla)
- GitHub Pages (pentru publicare)
